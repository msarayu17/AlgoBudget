# 💰 AlgoBudget – Student Expense & Budget Tracker

AlgoBudget is a full-stack web application designed to help students manage their finances efficiently. It enables users to create a budget, record daily expenses, visualize spending patterns, and monitor their financial health through an intuitive dashboard.

---

## 📌 Features

* 🔐 User Authentication (Sign Up & Login)
* 💵 Set and Manage Monthly Budget
* ➕ Add Daily Expenses
* 📅 Interactive Expense Calendar
* 📊 Expense Analytics with Charts
* 📋 Transaction History
* 👤 User Profile
* ⚙️ Settings Page
* 📱 Responsive User Interface
* 💾 Data Storage with MySQL

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AlgoBudget.git
```

### 2. Navigate to the Project

```bash
cd AlgoBudget
```

### 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

### 4. Start the React Application

```bash
npm run dev
```

### 5. Start the Flask Backend

Open a new terminal:

```bash
cd backend
pip install -r requirements.txt
python app.py
```

The frontend and backend should now be running locally.
